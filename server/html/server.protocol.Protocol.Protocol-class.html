handle(self, fno)

source code 

Obsługuje dane wysłane DO serwera.

Dane rozpoznawane są za pomocą headerów pakietów. Po więcej informacji na temat budowy pakietu należy przeczytać plik PACKETS dołączony do kodu źródłowego.

push_error(self, fno, head)

source code 

Klasa wysyłająca elementarne pakiety na temat błędów.

Konwencja jest następująca: Są to pakiety nie zawierające danych i posiadające header 0xE[numer]. Nie jest to jednak żadna reguła.
